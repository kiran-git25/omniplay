# Omni-Play
A privacy-first media and archive player. Runs 100% in browser
A privacy-first, browser-based media and archive player.

## Features
- Works fully offline
- No data ever leaves your device
- Supports video, audio, images, zip/rar archives, PDF, DOCX, XLSX, etc.
- Built with React, TypeScript, Tailwind, and Vite

## How to Run
```bash
npm install
npm run dev
